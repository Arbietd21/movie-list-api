URL Endpoints

Logic URL HTTP Method Request Body Response Body
Return a list of all movies. /movies GET none A JSON with a list of movies
Return data about a single movie by title. /movies/:movieTitle GET none A JSON with a information about a specific movie
Return data about a genre by name. /movies/genres/:genreName GET none A list of movies belonging to the corresponding genre.
Return data about a director by name. /movies/directors/:directorName GET none A JSON with information about the correspoding director.
Allow new users to register. /users POST { Username: req.body.Username,
Password: req.body.Password,
Email: req.body.Email,
Birthday: req.body.Birthday }		 JSON of the user that was just added
Allow users to update their user info(username). /users/:username PUT { Username: req.body.Username,
Password: req.body.Password,
Email: req.body.Email,
Birthday: req.body.Birthday }		 An updated JSON of the user and their updated info
Allow users to add a movie to their list of favorites. /users/:username/movies/:MovieID POST none An array of the users favorite movies
Allow users to remove a movie from their list of favorites. /users/:username/movies/:MovieID DELETE none Text confirming that a movie was successfully deleted from the users favorites list.
Allow an existing user to deregister. /users/:username DELETE none Text confirming a user was successfully deregistered
